# Clother

## Install Node and Angular

```bash
curl -sL https://deb.nodesource.com/setup_15.x | sudo -E bash -
sudo apt-get install -y nodejs

sudo npm install -g @angular/cli
```

## Clone from "dev" branch
```bash
git checkout dev
git fetch
```

## Inside /clother-angular
```bash
npm i
```

## Run local server 
```bash
ng serve --open
```
